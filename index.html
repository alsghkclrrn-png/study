⚠️ 성조 변화 규칙 (变调)

표기는 원래 성조 그대로 쓰지만, 실제 발음은 바뀝니다. 아래 규칙은 중국어 발음에서 가장 중요한 부분이에요.

① 3성 + 3성 → 2성 + 3성
3성이 연달아 두 개 오면, 앞의 3성은 발음하기 어렵기 때문에 2성으로 바뀝니다. 표기(ǎ ǎ)는 그대로지만 입으로는 á ǎ로 발음해요.
你好 nǐ hǎo → 실제 ní hǎo (안녕하세요)
很好 hěn hǎo → 실제 hén hǎo (아주 좋다)
水果 shuǐ guǒ → 실제 shuí guǒ (과일)
② 3성이 여러 개 연속될 때
3성이 셋 이상 이어지면 의미 단위로 끊어서 변조합니다. 보통 마지막 3성만 남기고 앞은 모두 2성처럼 발음돼요.
我很好 wǒ hěn hǎo → wó hén hǎo (저는 잘 지내요)
展览馆 zhǎn lǎn guǎn → zhán lán guǎn (전시관)
③ 반3성 (半三声)
3성 뒤에 1·2·4성이나 경성이 오면, 3성은 끝의 "올라가는 부분"을 생략하고 낮게 내리기만 합니다. 이걸 반3성이라고 해요. 회화에서 3성은 대부분 이렇게 발음돼요.
老师 lǎo shī (3성+1성) → 老를 낮게만
语言 yǔ yán (3성+2성) · 可是 kě shì (3성+4성)
④ 不 (bù)의 변조
不는 원래 4성(bù)이지만, 뒤에 4성이 오면 2성(bú)으로 바뀝니다.
不是 bù shì → bú shì (아니다) — 뒤가 4성
不好 bù hǎo (안 좋다) — 뒤가 3성이라 그대로 4성
不去 bù qù → bú qù (안 간다) — 뒤가 4성
⑤ 一 (yī)의 변조
一는 원래 1성(yī)이지만 위치에 따라 바뀝니다.
· 단독·서수·끝 → 그대로 1성 (一 / 第一 / 星期一)
· 뒤에 1·2·3성 → 4성(yì)
· 뒤에 4성 → 2성(yí)
一天 yì tiān (하루) — 뒤 1성 · 一起 yì qǐ — 뒤 3성
一个 yí gè (한 개) — 뒤 4성 · 一样 yí yàng (같다) — 뒤 4성
🔤 발음(병음) 핵심 포인트
한국인이 헷갈리는 자음
b / pb는 된소리(ㅃ)에 가깝고, p는 강한 기식음(ㅍ+숨)
d / td는 ㄸ에 가깝고, t는 ㅌ에 바람을 더 세게
g / kg는 ㄲ, k는 ㅋ에 바람을 강하게
j / q / x혀를 입천장 앞에. j=ㅈ, q=ㅊ(기식), x=ㅅ과 ㅆ 사이
zh / ch / sh / r혀를 말아 올림(권설음). 한국어에 없는 소리
z / c / s혀끝을 윗니 뒤에. z=ㅉ, c=ㅊ, s=ㅆ
주의할 모음·운모
ü (yu)입을 'ㅜ' 모양으로 오므린 채 'ㅣ' 발음. 한국어에 없음
e단독 e는 'ㅓ'와 'ㅡ' 중간 소리
er혀를 말며 '얼'. 儿 érzi 등
-i (zhi,chi,shi)'이'가 아니라 혀를 만 상태의 'ㅡ'에 가까움
ian'이안'이 아니라 '이엔'에 가깝게
ün / üej,q,x,y 뒤의 u는 사실 ü 발음 (ju=쥐, xu=쉬)
